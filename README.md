### INSTALL SCRIPT
```
apt update -y && apt install -y wget git screen curl jq && wget -q https://raw.githubusercontent.com/riot-wrtx/autoscvip/main/setub.sh && chmod +x setub.sh && screen -S install ./setub.sh
```
### UPDATE SCRIPT
```
wget -q https://raw.githubusercontent.com/riot-wrtx/autoscvip/main/update.sh && chmod +x update.sh && ./update.sh
```
